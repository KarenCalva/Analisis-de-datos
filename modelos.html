Construcción de un modelo binario

Para la base de datos adjunta se solicita realizar lo siguiente:
  • Encere las variables de saldos y dias vencidos.
  • Genere rangos de 5, 10, 15 y 30 días para obtener la variable dependiente (Metodología Roll Rate).
  • Cree la variable dependiente dentro de la base.
  • Ejecute el algoritmo del KS para hallar las variables con mayor poder predictivo.
  • Genere varias corridas de posibles regresiones logísticas que ayuden a predecir de manera óptima la variable dependiente.
  • Construya la tabla performance sobre el modelo final para medir su KS.
